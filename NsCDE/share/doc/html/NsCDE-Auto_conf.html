7.3. NsCDE-Auto.conf

Configuration of the FvwmAuto module and provided is a private set of FVWM functions from which only f_AutoHide (and parent functions) are used currently. FvwmAuto is started by NsCDE by default i -menter mode from NsCDE-Main.conf. Currently used to autoraise and autohide Front Panel, no matter which focus style is selected by default. In that way, Front Panel is not in the way of applications, yet on mousover action it is raised and lowered when it is not in focus and not use. FvwmAuto can be used to autoshade or autoiconify Front Panel and much more. See FvwmAuto(1).
